## Overview

The applications system in LVLX allows youth to apply for vacancies and employers to manage these applications efficiently. This guide covers both the youth and employer perspective of the application process.

## For Youth/Job Seekers

### How to Apply for a Vacancy

1. Navigate to the vacancy details page
2. Click the "Apply" button
3. Select your resume:
   ```
   - Choose an existing resume
   - Or create a new resume (File or Custom type)
   ```
4. Submit your application

### Track Your Applications

1. Go to "My Applications" dashboard
2. View application status:
   ```
   Pending: Application is under review
   Shortlisted: Selected for further consideration
   Interview: Invited for an interview
   Rejected: Not selected for the position
   Hired: Successfully selected for the position
   ```

### Application Tips

1. Keep your resume updated
2. Check application status regularly
3. Respond promptly to interview invitations
4. Maintain professional communication


## For Employers

### Managing Applications

1. Access "Applications Dashboard"
2. View all applications for your vacancies:
   ```
   Filter by:
   - Vacancy
   - Status
   - Date received
   ```

### Processing Applications

1. Review application details:

   ```
   - Candidate profile
   - Resume
   - Application date
   ```

2. Update application status:
   ```
   Pending → Initial state
   Shortlisted → Candidate under consideration
   Interview → Schedule an interview
   Rejected → Not selected
   Hired → Position offered and accepted
   ```

### Interview Management

1. Schedule interviews:

   ```
   - Set date and time
   - Choose interview type (Online/Physical)
   - Add location or meeting link
   - Include interview description
   ```

2. Track interview status:
   ```
   - Pending interviews
   - Completed interviews
   - Cancelled/Rescheduled
   ```

### Best Practices

1. Review applications promptly
2. Maintain clear communication
3. Update application status regularly
4. Document interview feedback
5. Follow up with candidates

## Points System

### Cost Structure

Applications management involves point usage:

```
- Viewing candidate details: 1 point
- Scheduling interviews: 5 points
- Sending invites: 1 point
```

### Managing Points

1. Monitor point balance
2. Purchase points as needed
3. Track point usage in transactions

## Privacy and Data Protection

1. Application data is securely stored
2. Access is restricted to authorized personnel
3. Personal information is protected
4. Data retention policies apply
