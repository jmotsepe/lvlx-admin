## Overview

This guide covers the administrative functions for managing companies on LVLX. Access to these features is restricted to users with the Admin role.

## Company Management Dashboard

1. Main Dashboard Features:
   ```
   - List of all verified companies
   - Company details (name, department, managers count)
   - Vacancy counts per company
   - Company creation dates
   - Quick access to company profiles
   ```

2. Key Information Displayed:
   ```
   - Company name
   - Company owner details
   - Department
   - Number of managers
   - Number of vacancies
   - Creation date
   ```

## Company Verification Process

1. Accessing Pending Companies:
   ```
   - View unverified companies count
   - Access pending companies list
   - Review company details
   ```

2. Verification Review:
   ```
   Required Documents:
   - Registration certificate (PDF format)
   - Company registration number
   - Physical address
   - Employee count
   - Company type
   ```

3. Verification Actions:
   ```
   - View company details
   - Download and verify registration certificate
   - Approve company
   - Reject company
   ```

## Access Control

1. Admin Restrictions:
   ```
   - Only Admin role can access
   - Automatic redirect for non-admins
   - Protected verification actions
   ```

2. Company Status Management:
   ```
   Available Statuses:
   - Pending: Awaiting verification
   - Approved: Verified and active
   - Rejected: Verification declined
   ```

## Best Practices

1. Verification Guidelines:
   ```
   - Always check registration certificate
   - Verify company registration number
   - Confirm business legitimacy
   - Review all required information
   ```

2. Decision Making:
   ```
   - Do not approve without valid certificate
   - Check registration number format
   - Verify physical address
   - Ensure complete profile
   ```
