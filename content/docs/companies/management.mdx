## Team Management

### Adding Team Members

1. Invite Process:
   ```
   - Navigate to "Team Management"
   - Click "Add Manager"
   - Enter manager's email
   - Select role and permissions
   - Send invitation
   ```

### Manager Roles

1. Available Positions:
   ```
   - Primary Manager (Company Owner)
   - Department Manager
   - Recruitment Manager
   - Points Manager
   ```

2. Role Permissions:
   ```
   Primary Manager:
   - Full company control
   - Add/remove managers
   - All system access

   Department Manager:
   - Manage specific departments
   - Handle team members
   - View department metrics

   Recruitment Manager:
   - Post vacancies
   - Review applications
   - Schedule interviews

   Points Manager:
   - Monitor points balance
   - Purchase points
   - Allocate resources
   ```
