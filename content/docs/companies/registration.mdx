## Overview

This guide covers the step-by-step process of registering your company on LVLX, from initial submission to verification.

## Initial Registration

1. Navigate to Companies:
   ```
   - Click "Companies" in main navigation
   - Select "Add New Company"
   ```

2. Complete Registration Form:
   ```
   Required Information:
   - Company Name
   - Department
   - Physical Address
   - Registration Number (Format: YYYY/NNNNN/LL)
   - Employee Count
   - Company Type (Private, Public, or Organisation)
   - Company Description
   ```

3. Document Requirements:
   ```
   Registration Certificate:
   - PDF format only
   - Maximum size: 10MB
   - Must be clear and legible
   - Official company registration document
   ```

## Verification Process

1. After Submission:
   ```
   - Application enters verification queue
   - Status changes to "Pending"
   - Estimated review time: 24-48 hours
   ```

2. During Verification:
   ```
   - Documents are reviewed
   - Company details validated
   - Registration number checked
   - Business legitimacy confirmed
   ```

3. Track Progress:
   ```
   Dashboard Indicators:
   - Pending: Under review
   - Approved: Verification successful
   - Rejected: Verification failed
   ```

## Common Registration Issues

1. Document Problems:
   ```
   - File too large
   - Wrong format
   - Unclear scan
   - Expired certificate
   ```

2. Information Errors:
   ```
   - Invalid registration number
   - Incomplete address
   - Missing department info
   - Incorrect company type
   ```

## Best Practices

1. Before Registration:
   ```
   - Prepare all documents
   - Check file formats
   - Verify company details
   - Review requirements
   ```

2. During Registration:
   ```
   - Double-check all entries
   - Use official information
   - Provide accurate contact details
   - Save progress regularly
   ```
