## Overview

The invites system allows employers to invite potential employees or team members to join their company on LVLX. This feature streamlines the process of building your team and managing company access.

## For Employers

### Sending Invites

1. Navigate to the Invites dashboard
2. Click "Invite User" button
3. Fill in the invite details:
   ```
   - Name: The invitee's full name
   - Company: Select your company
   - Email: Invitee's email address
   - Message: Optional personalized message
   ```
4. Click "Send Invitation"

### Managing Invites

1. View all sent invites on the dashboard
2. For each invite, you can:
   ```
   - See invite status
   - View when it was sent
   - Remove/Cancel the invite
   ```

### Points System

1. Cost per invite:
   ```
   Sending an invite: 1 point
   ```
2. Requirements:
   ```
   - Sufficient point balance
   - Active company status
   ```

### Best Practices

1. Personalize invite messages
2. Verify email addresses before sending
3. Follow up with invitees
4. Remove unused or expired invites
5. Keep track of your invites quota

## For Invitees

### Accepting an Invite

1. Receive invite email containing:
   ```
   - Company information
   - Sender's name
   - Personal message
   - Unique invite code
   ```
2. Click the invitation link
3. Enter the provided code
4. Complete registration process

### After Accepting

1. Set up your profile
2. Connect with your company
3. Access company-specific features

## Security Features

1. Unique codes:

   ```
   - 6-digit secure codes
   - One-time use
   - Limited validity period
   ```

2. Email verification:
   ```
   - Automated email confirmation
   - Secure code delivery
   - Anti-spam measures
   ```

## Important Notes

1. One invite per email address
2. Points are deducted upon sending
3. Automatic notifications for:
   ```
   - Invite sent
   - Invite accepted
   - Invite removed
   ```
