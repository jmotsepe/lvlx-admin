## Overview

The profile management system in LVLX allows users to maintain their personal information, account settings, and preferences. This comprehensive guide covers all aspects of profile management for different user roles.

## Personal Information Management

### Basic Profile Details

1. Access your profile settings:

   ```
   Dashboard → Profile → Account
   ```

2. Edit personal information:

   ```
   - First Name
   - Last Name
   - Email Address
   - Cell Number
   - Gender
   - Province
   ```

3. Gender options:

   ```
   - Male
   - Female
   - Non-Binary
   - Other
   ```

4. Province selection:
   ```
   - Gauteng
   - Limpopo
   - North West
   - Free State
   - KwaZulu-Natal
   - Mpumalanga
   - Northern Cape
   - Eastern Cape
   - Western Cape
   ```

### Profile Status

Your profile can have different statuses:

```
Pending: Initial state after registration
Approved: Verified and active profile
Rejected: Profile needs attention
```
